## 🏵 Introduction

**Inspection** is a debug accessory tool for iOS.

## 📋 Requirements

- iOS 9.0+
- Xcode 9.3+ with Swift 4.1+

## 📲 Installation

`Inspection` is available on [CocoaPods](https://cocoapods.org/):

```ruby
use_frameworks!
pod 'Inspection'
```
